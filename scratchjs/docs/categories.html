List of Categories

The extension provides many different categories of blocks. Here are the main groups:

Data and Storage

Categories: Arrays, Objects, Storage, Temporary Variables

Store and organize data using arrays and objects, save data between project runs, and work with temporary variables.

Math and Logic

Categories: Math, Constants, Booleans

Advanced math operations, useful constants, and boolean logic for making decisions in your code.

Text and Strings

Categories: Strings, Unicode

Text manipulation blocks for splitting, joining, replacing, and working with text in various ways.

Time and Timing

Categories: Date and Time

Get current date and time, work with timestamps, create delays and timers for your projects.

Browser and Web

Categories: Browser, Color

Control the browser window, handle user input, work with colors, and interact with web pages.

Games and Fun

Categories: Games, Statistics

Blocks for scoring, statistics tracking, and common game mechanics.

Development Tools

Categories: Console, Core JS Blocks, Utilities

Debugging tools, core JavaScript functionality, and utility blocks for common programming tasks.
