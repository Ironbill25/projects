JSON Documentation

JSON (JavaScript Object Notation) is a way to store and organize data. ScratchJS provides blocks to work with Arrays and Objects.

Arrays

Arrays are ordered lists of items. Think of them like a list of grocery items.

You can create arrays using the "Blank Array" block or the "Split () by ()" block.
There are various blocks for things like to add items to the start or end, remove items from any position, or get items by their position number.
While most arrays are 0-based (the first item is index 0), ScratchJS arrays are 1-based (the first item is index 1).

The Arrays category also has blocks to get length, check if it contains something, join items into text, and sort them.

Objects

Objects are collections of key-value pairs. Think of them like a dictionary where you look up a word (key) to find its definition (value).

You can create objects using the "create object" block, get values using keys, set or change values, and work with keys.
Objects in ScratchJS are stored as plain text as Scratch doesn't currently support native object types.

Remember: Arrays use numbers (indexes) to access items, while objects use text (keys) to access values.
