ScratchJS Documentation

Welcome to the ScratchJS documentation! Here you can learn how to use the extension and its blocks.

Getting Started

ScratchJS adds new blocks to Scratch that let you do more with your projects.

Usage

There are two ways to use ScratchJS. You can use a bookmarklet, or copy and paste the code.
The bookmarklet is the easiest option. You can find it on the main page.
You can also copy and paste the code from the main page into a new bookmark if the link doesn't work.

Please Note: The bookmarklet only works on Scratch project pages, not the main Scratch website.

Using the Blocks

After running the bookmarklet, you will see a new category in the editor titled "ScratchJS".
The sub-categories are grouped by functionality for convenience.

Main Article: All Categories

Categories

The extension provides many different categories of blocks. Here are the main groups:

Data and Storage

Categories: Arrays, Objects, Storage, Temporary Variables

Store and organize data using arrays and objects, save data between project runs, and work with temporary variables.

Math and Logic

Categories: Math, Constants, Booleans

Advanced math operations, useful constants, and boolean logic for making decisions in your code.

Text and Strings

Categories: Strings

Text manipulation blocks for splitting, joining, replacing, and working with text in various ways.

Time and Timing

Categories: Date and Time, Timing

Get current date and time, work with timestamps, create delays and timers for your projects.

Browser and Web

Categories: Browser, Input, Color, Styles

Control the browser window, handle user input, work with colors and CSS styles, and interact with web pages.

Games and Fun

Categories: Games, Statistics

Game-specific blocks for scoring, statistics tracking, and common game mechanics.

Development Tools

Categories: Console, Core JS Blocks, Utilities

Debugging tools, core JavaScript functionality, and utility blocks for common programming tasks.

Frequently Asked Questions

Why isn't ScratchJS loading?

There is a current known issue with the bookmarklet not loading on the first visit to a project page.
Try refreshing the page and clicking the bookmarklet again. If this doesn't work, check your internet and settings.
If you cannot get it to work, make sure that javascript: URLs are allowed in your browser.

Warning: Some schools or networks block bookmarklets. Ask your teacher or administrator for help.
Also, if you are at school, make sure you have your teacher's permission to use this extension as it could potentially do things like bypass filters.

Why don't the blocks work?

If the blocks appear but don't work, try refreshing the project page and clicking the bookmarklet again.
Make sure that you are using the blocks correctly and that you have the latest version of ScratchJS.
If the issue persists, check the browser console for error messages.

Can I use this in my school project?

Yes, ScratchJS is free to use for any project. Make sure your teacher allows it, though.

Note: Make sure that you have permission to use this extension. Some networks also block certain websites that this extension needs to access.

How do I report bugs?

You can report bugs and issues on the GitHub issues page. Make sure to include the following:

  • What you were trying to do,
  • What happened instead,
  • Any error messages,
  • Screenshots or videos, if possible,
  • and your browser and computer type.
